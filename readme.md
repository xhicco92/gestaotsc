# 🚀 Dashboard Centro Técnico

Dashboard para gestão e monitoramento do centro técnico, desenvolvido para visualização de OS abertas e métricas de produtividade da equipe.

## 📸 Preview

![Dashboard Preview](https://via.placeholder.com/800x400/4facfe/ffffff?text=Dashboard+Centro+Técnico)

## ✨ Funcionalidades

- **📋 Abertos Área Técnica**: Visualização das ordens de serviço em aberto
- **⚡ Produtividades**: Dashboard completo com métricas de desempenho
  - OS concluídas por período
  - Horas trabalhadas
  - Produtividade média da equipe
  - Técnicos ativos
  - Filtros por período (Hoje, Ontem, Semana, Mês, Trimestre, Ano)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com Flexbox e Grid)
- JavaScript (ES6+)
- Design Responsivo
- CSS Animations

## 📋 Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Git (para clonar o repositório)
